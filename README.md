## 🚀 Running the project

### Back-end

Clone the project

```bash
  git clone https://github.com/skmonsterr/nlwcopa
```

Enter the project directory

```bash
  cd server
```

Install with dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```

### Front-end Web

Clone the project

```bash
  git clone https://github.com/skmonsterr/nlwcopa
```

Enter the project directory

```bash
  cd web
```

Install with dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```

### Mobile

